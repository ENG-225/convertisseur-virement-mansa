\# 🏦 Convertisseur de Fichier de Virement - MANSA BANK



Application de conversion automatique des fichiers de virement au format standardisé pour les opérations bancaires.



\## ✨ Fonctionnalités



\- 📁 \*\*Chargement de fichier\*\* : Glisser-déposer ou saisie du chemin complet

\- 🔄 \*\*Conversion automatique\*\* : Traitement des RIB, montants et motifs

\- 👁️ \*\*Aperçu\*\* : Visualisation des données avant téléchargement

\- 💾 \*\*Téléchargement\*\* : Fichier Excel formaté prêt à l'emploi
